# Stone Stick By Nano

A Button-based locomotion for RUMBLE Which Removes All Analog Stick Controls For Analog Stick Drift Users.

## Controls
- X: forward
- Y: left
- A: backward
- B: right

## Installation (MelonLoader)
1. Install MelonLoader for RUMBLE.
2. Copy `StoneStickByNano.dll` into `RUMBLE/Mods`.
